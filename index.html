Getting Started

  • Tutorial avaliable at http://fsharp.github.io/FAKE/
  • Download via NuGet
@echo off
cls
.nuget\NuGet.exe Install FAKE -OutputDirectory packages -ExcludeVersion
packages\FAKE\tools\Fake.exe build.fsx
pause

or

@echo off
cls
.paket\paket.exe restore
packages\FAKE\tools\Fake.exe build.fsx
pause
